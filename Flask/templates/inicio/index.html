Rutas JSON

  • /listaAlumnos - devuelve una lista de los alumnos en formato JSON
  • /listaMaterias- devuelve una lista de las materias en formato JSON
  • /listaEventos- devuelve una lista de los eventos formato JSON
  • /alumnos/agregar- agrega un alumno por peticion POST
  • /alumnos/eliminar- elimina un alumno por medio del ID en el header (id_alumno)
  • /materias/agregar- agrega una materia por peticion POST
  • /materias/eliminar- elimina una materia por medio del ID en el header (id_materia)
  • /eventos/agregar- agrega un evento por peticion POST
  • /eventos/eliminar- elimina un evento por medio del ID en el header (id_materia)
  • /evento/findById/:id - devuelve la informacion de un evento en formato JSON si es encontrada, en caso de que no, devuelve un error en formato JSON
